Tic-Tac-Toe Game using Java Swing

This is a desktop-based GUI application that allows two players to play the classic Tic-Tac-Toe game. It is developed using Java Swing for the graphical user interface. The game features a 3x3 grid, handles player turns, determines the winner or a draw, and visually highlights the result.

Tools & Technologies Used
  • Programming Language: Java
  • IDE: Eclipse / IntelliJ IDEA / NetBeans (any preferred)
  • GUI Toolkit: Java Swing
  • JDK Version: Java SE 8 or later
  • To implement a classic Tic-Tac-Toe game using Java's GUI toolkit.
  • To understand and demonstrate event-driven programming using ActionListener.
  • To create a responsive and interactive user interface.
  • To develop logical thinking by implementing game rules (turns, win conditions, draw).
  • To gain experience with 2D arrays, loops, conditional logic, and basic OOP concepts.
