Title: AI-based Disease Diagnosis and Prescription System

MediAssist is an AI-integrated medical web application built with Django that allows users to receive real-time consultation by chatting with a virtual doctor. Based on the selected disease, the AI doctor asks relevant questions, analyzes user responses, and finally provides a downloadable prescription.

Tools & Technologies Used
  • Backend: Django (Python)
  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • AI Engine: Integrated question-handling and response logic
Goals & Objectives
  • Provide an automated, accessible, and smart healthcare assistant.
  • Enable users to chat with an AI doctor based on their disease selection.
  • Collect symptoms and intelligently generate a prescription.
  • Allow users to download the prescription for further use
  • Let admins dynamically manage diseases through an admin panel.
User Flow / Journey Map
  • User Login & Registration
    • Secure login system with user profile support.
  • Disease Selection.
    • Users select a disease category (e.g., Diabetes, Fever, COVID-19) to begin the session.
  • SAI Chat System.
    • The virtual doctor interacts with the user, asks relevant questions, and collects symptom data.
  • Automated Prescription Generation
    • Based on responses, the system dynamically generates a prescription in PDF format.
  • Admin Panel
    • Admin can add/edit/delete diseases and update question logic without modifying the code.
Challenges & Solutions
  • Designing smart AI conversation flow for multiple diseases.
  • Dynamic prescription generation Implemented Django-based PDF generator for personalized outputs.
  • Built an admin dashboard to manage disease and question logic dynamically
Future upgrades could include
  • Add multi-language support
  • Incorporate appointment scheduling with real doctors
  • Enhance AI model with machine learning for smarter diagnosis
