YouTube Audio to Text Converter

YT Audio Transcriber is a web application that allows users to paste a YouTube video link, convert the audio into text using speech recognition, and download both the audio file (MP3) and transcribed text (PDF). This project was built to help users easily extract and save important content like lectures, interviews, or tutorials.

Tools & Technologies Used
  • Backend: Django (Python)
  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • YouTube Download: pytube or youtube_dl
  • Audio Conversion: ffmpeg
  • Speech Recognition: SpeechRecognition, pydub
  • PDF Export: xhtml2pdf
Goals & Objectives
  • Allow users to extract audio from YouTube videos.
  • Use speech-to-text technology to transcribe the audio.
  • Enable users to download both audio and transcription.
  • Ensure a clean and user-friendly experience with minimal steps.
Features
  • User Authentication
    • Secure signup and login system to manage access.
  • YouTube Link Input
    • After login, users can paste any YouTube video link.
  • Audio Extraction & Conversion
    • Automatically converts YouTube video to MP3 format.
  • Speech-to-Text Transcription
    • Extracts spoken content from the video and converts it to readable text.
  • Download Options
    • MP3 file of the audio
    • PDF document of the transcribed text
User Flow / Journey Map
  • User registers or logs in to the platform.
  • User pastes a YouTube video URL.
  • System downloads and converts the video to audio.
  • Audio is processed and transcribed to text.
  • User is provided with download buttons for: The MP3 audio & The PDF transcription
Future upgrades could include
  • Add language selection and translation support
