Pizza Store - UI Design

Title: A Deliciously Designed Pizza Store Website.

a modern and interactive pizza store website that lets users browse pizzas, explore offers, read about the store’s history and chefs, and discover store locations. While anyone can explore the site without logging in, only registered users can place orders or book a table online.
The goal was to design a clean, fun, and engaging user experience that feels as satisfying as the pizzas it promotes.

Tools & Technologies Used
  • Figma
Goals & Objectives
  • Let users freely explore menu items and offers
  • Encourage user registration for online ordering and table booking
  • Build trust and curiosity through detailed “About Us” and “Our Chefs” sections.
  • Enable users to contact the store or subscribe to a newsletter.
Features
  • Navigation Bar & Hero Section
    • Top navbar with menus, Login / Signup button
    • Full-width banner with delicious pizza image
  • Menu Page & Offers Page
    • Users see in menu page category wise all items
    • Active offers and deals
  • About Us Page & Locations Page
    • Interactive map
    • List of all branch addresses and contact numbers
    • Simple form with Name, Email, Subject, Message
  • Footer Section & Newsletter Section
    • Quick links to all pages
    • Social media icons
    • Engaging the User with Newsletter Signup
  • Login/Signup Flow
    • Users can browse freely
    • To order pizza or book a table, login/signup is required
User Flow / Journey Map
  • A[Homepage] --> B[Explore Menu/Offers/About]
  • B --> C{Wants to Order or Book?}
  • C -- Yes --> D{Logged In?}
  • D -- No --> E[Prompt Login/Signup]
  • E --> F[Redirect to Auth Page]
  • D -- Yes --> G[Proceed to Order/Table Booking]
  • C -- No --> H[Continue Browsing]
Future upgrades could include
  • Let users rate & review pizzas
  • In-app chatbot for ordering
