Restro – Restaurant UX Design

Title: Restro – A Seamless Restaurant UX for Browsing, Booking & Ordering

Restro is a full-featured restaurant website that combines elegant UI with smooth user experience. The goal was to design a customer-friendly platform where users can explore the restaurant’s ambiance, menu, history, and easily order food or book tables — all while ensuring user authentication for secured actions.

Tools & Technologies Used
  • Figma
Goals & Objectives
  • Provide a rich and intuitive browsing experience for food lovers
  • Ensure user login/signup for secure booking & ordering
  • Build customer trust through the “About Us” section.
  • Allow users to ask questions and send queries via “Contact Us”.
  • Collect user feedback to improve service.
  • Deliver a well-structured, modern footer for navigation and trust.
Features
  • Home Page (Explore First)
    • Eye-catching hero section with Order Now, View menu, About hours buttons.
    • Browse menu, offers, chef’s specials
  • Login / Signup
    • Online food ordering
    • Table reservation
    • Clean and simple UI with password visibility toggle and validation
    • Social login option for faster access (Google, Facebook)
  • Ordering System
    • Filter by category (Starters, Main, Dessert, Drinks).
    • Add to cart, quantity selector
    • Order summary & checkout
  • Table Booking
    • Pick a date, time, number of guests.
    • Check availability in real-time
  • Contact Us
    • Easy-to-fill form
    • Acknowledgement message after submission.
  • About Us & Feedback Section
    • Tells the story of the restaurant
    • Logged-in users can leave reviews.
    • Useful for continuous improvement
  • Footer Design
    • Newsletter Subscription
    • Social Media Links (Facebook, Instagram, Twitter)
    • Policy Links (Privacy, Terms & Conditions)
    • Quick Links (Home, Menu, Order, Contact)
User Journey Map

A[Explore Website] --> B[Check Menu/About/Contact]

B --> C{Wants to Order/Book?}

C -- Yes --> D[Prompt to Login/Signup]

D --> E[Order Food or Book Table]

C -- No --> F[Continue Browsing or Ask Question]

E --> G[Get Confirmation]

F --> H[Submit Query via Contact Us]

Future upgrades could include
  • Live chat integration for instant help
  • Dark mode toggle for accessibility.
