CoffeeShop – Your Personalized Coffee Experience (UI Design)

Title: CoffeeShop – A Mobile App for Exploring and Ordering Premium Coffee.

CoffeeShop is a mobile application designed to offer users a rich and smooth coffee ordering experience. The app provides easy onboarding, personalized coffee recommendations, category-wise browsing, profile management, and a complete coffee ordering system.

Tools & Technologies Used
  • Figma
Goals & Objectives
  • Create a clean and user-friendly interface.
  • Allow both new and returning users to log in seamlessly.
  • Display category-wise and rating-wise coffee options.
  • Allow profile updates and order management.
  • Build brand trust through sliders, chefs, partners, and newsletter.
Problem Statement

Most coffee apps either overload the user or lack intuitive flow. The challenge was to:

  • Simplify login/signup with verification
  • Personalize the user experience
  • Make coffee discovery easy
User Flow / Journey Map
  • App Launches → Splash Screen with CoffeeShop Logo.
  • Login / Signup screen with verification code.
  • After login → Home Page:
    • Slider with featured content
    • Search bar to find specific coffee
    • Category-wise coffee list (e.g., Espresso, Latte, Cold Brew)
    • Rating-wise best coffee
    • “Our Chefs” and “Our Partners” section
    • Newsletter subscription
  • Bottom Navigation:
    • Home
    • Menu (complete coffee list, filters by category/rating, search)
    • Profile (update info)
  • Order Process:
    • Add coffee to cart
    • Confirm order
Future upgrades could include
  • Push notifications for offers
  • Loyalty rewards
